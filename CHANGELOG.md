# Changelog

All notable changes to the Person Bio desktop extension are documented here.

## [1.0.0] — 2026-07-05

Initial release.

- Remote-MCP bridge to `https://person.bio/api/v1/mcp` via bundled `mcp-remote` 0.1.38.
- OAuth sign-in (no secrets stored in the extension).
- Résumé, life-journey, self-intro, blog, and job tools — each annotated
  read-only or write, with per-call consent.
- MIT licensed; manifest `author` points at the GitHub profile.
